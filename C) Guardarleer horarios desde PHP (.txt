C) Guardar/leer horarios desde PHP (tutor/mentor)

En addSchedule({...}) cámbialo por llamar al backend:

async function addSchedule({owner, type, day, start, end, subjects=[]}){
  const body = { ownerId:owner.id, ownerRole:owner.role, ownerName:owner.name||'', email:owner.email||'', type, day, start, end, subjects };
  const r = await api('/schedules.php', { method:'POST', body: JSON.stringify(body) });
  return r.id;
}


En tablas “Mis horarios”:

renderTutorTable(user) → usar GET /schedules.php?ownerId=RFC

Eliminar → DELETE /schedules.php?id=ID&actorRole=tutor&actorId=RFC

Igual para mentor con actorRole=mentor.

Ejemplo de tabla tutor:

async function renderTutorTable(user){
  const wrap = $('#tablaTutor');
  const my = (await api('/schedules.php?ownerId=' + encodeURIComponent(user.id))).items || [];
  if(!my.length){ wrap.innerHTML = '<p class="text-sm text-slate-700">Sin horarios registrados.</p>'; return; }
  wrap.innerHTML = tableSchedules(my, true);
  $$('#tablaTutor button[data-del]').forEach(btn => btn.onclick = async ()=>{
    const id = btn.dataset.del;
    await api(`/schedules.php?id=${encodeURIComponent(id)}&actorRole=tutor&actorId=${encodeURIComponent(user.id)}`, {method:'DELETE'});
    await renderTutorTable(user);
  });
}
